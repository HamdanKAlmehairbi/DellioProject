## Installation Instructions for User Interface:

git clone repository

cd into folder nextjs-interviewai

run the following command to install dependencies:

**npm install**

Run the following command to launch the app:

**npm run dev**

Open the following url in your browser:

**http://localhost:3000**


## Installation Instructions for Fast API Backend:

run the following commands to install dependencies for Fast API and MongoDB:

**pip install "fastapi[all]"**

**pip install pymongo**

cd into folder nextjs-interviewai/PythonBackend

Run the following command in a separate terminal to launch the backend:

**uvicorn main:app --reload**

You can go to the following url in your browser to view and test the endpoints independent of the frontend user interface:

**http://localhost:8000/docs**

Note: one terminal runs the frontend user interface and the other terminal runs the backend
